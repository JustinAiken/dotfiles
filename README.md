1. Install ZSH
2. Set as shell ```chsh -s `which zsh````
3. Install oh-my-zsh `git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh`
4.

```sh
git clone git@github.com:JustinAiken/dotfiles.git && \
chmod +x ~/dotfiles/sync.sh && \
~/dotfiles/sync.sh
```
